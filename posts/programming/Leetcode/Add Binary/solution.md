<h3> Add Binary </h3>

![image](https://github.com/h4ckyou/h4ckyou.github.io/assets/127159644/84b470ca-188f-467e-aaef-be2aa2bf525e)


Solve Script:

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        return bin(int(a, 2) + int(b, 2))[2:]
```
